# Student-Information-Management-System
Student Information Management System using JDBC and MySQL

📌 Project Overview
The Student Information Management System is a desktop-based application developed using Java Swing, JDBC, and MySQL. The system helps manage student records efficiently through a user-friendly graphical interface. It provides secure login authentication and supports complete CRUD (Create, Read, Update, Delete) operations.
